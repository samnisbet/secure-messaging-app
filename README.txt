How to Run our code:

First you will need the Expo framwork installed

https://docs.expo.dev/get-started/installation/

Once installed and the terminal is in the project directory, fetch any dependencies/modules needed for compilation. run expo start.

For web: press w
For Android: a
For IOS: either QR code or given input in console

Application should open on local host in web or just appear on mobile device/simulator.
